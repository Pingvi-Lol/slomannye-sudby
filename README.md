# Сломанные Судьбы — Сайт фанфика

## Файлы:
- index.html — главная страница (показывает серии)
- admin.html — админ-панель
- episodes.json — база данных серий
- style.css — стиль сайта

## Админ-панель
Пароль: admin123  
После входа можно:
- добавлять серию  
- редактировать  
- удалять  
- скачивать обновлённый episodes.json  

## Хостинг
Сайт можно загрузить на:
- GitHub Pages
- Netlify
- Vercel

Важно: каждый раз после изменения серий нужно загружать обновлённый episodes.json.
