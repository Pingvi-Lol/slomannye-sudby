# Сломанные судьбы — сайт фанфика

## Что внутри
- `index.html` — выбор сезона
- `season.html` — страницы сезона с сериями
- `admin.html` — админ панель (вход по паролю)
- `seasons.json` — начальные данные (можно импортировать)
- `style.css` — стиль сайта

## Как установить на GitHub Pages
1. Создай репозиторий на GitHub (имя любое или `username.github.io`).
2. Скопируй в корень репозитория файлы: `index.html`, `season.html`, `admin.html`, `style.css`, `seasons.json`.
3. Перейди в **Settings → Pages** и включи GitHub Pages (branch: main / root).
4. Страница будет доступна по `https://<username>.github.io/<repo>/` или `https://<username>.github.io/` если репо с таким именем.

## Как редактировать серии
- Открой `admin.html`
- Введи пароль `BrokenAdmin2024` (можно поменять в коде)
- Редактируй сезоны и серии
- Нажми **Экспорт JSON** чтобы скачать обновлённый `seasons_export.json`
- Загрузить экспортированный JSON в репозиторий как `seasons.json` или использовать `Импорт JSON` на другом устройстве

> Примечание: сейчас данные хранятся в `localStorage` в браузере.  
> Для публичного редактирования и автоматического обновления потребуется подключение к GitHub API и токену (это можно добавить позже).

## Блокировка сезонов
- Сезоны 3 и 4 помечены как `locked: true` и показывают сообщение «Сезон недоступен. Он выйдет позже.»

## Подсказки
- Пароль можно заменить в `admin.html` (константа `ADMIN_PASS`).
- Для резервной копии используйте кнопку `Экспорт JSON`.
